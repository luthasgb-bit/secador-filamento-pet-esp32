# Secador de Filamento PET con Control Ambiental – ESP32

# Descripción general
Este proyecto corresponde al Trabajo Final de la **Diplomatura en Robótica Educativa**.
Consiste en el diseño y desarrollo de un **módulo de secado y acondicionamiento de filamento PET**, orientado a contextos educativos maker, con criterios de eficiencia energética, reutilización de materiales y control electrónico.

El sistema permite controlar temperatura y humedad dentro de un gabinete cerrado, utilizando un microcontrolador ESP32 y sensores ambientales, con el objetivo de mejorar la calidad del filamento reciclado para impresión 3D.

---

# Plataforma de hardware
- Microcontrolador: ESP32 Dev Module
- Sensores: temperatura y humedad (ej. DHT22)
- Actuadores: resistencia calefactora de bajo consumo, ventilador 12V
- Sistema modular: portacarrete inferior + módulo de control superior
- Gabinete fabricado con chapas recicladas de CPU

---

# Lenguaje y entorno de desarrollo
- Lenguaje de programación: **C++ (Arduino Framework)**
- Tipo de programación: **texto**
- Entorno de desarrollo: Arduino IDE
- Sistema operativo compatible: Windows / Linux

---

# Fundamentación técnica
La solución se basa en el uso de componentes estándar, de bajo costo y ampliamente documentados.
El ESP32 fue elegido por su capacidad de procesamiento, conectividad y escalabilidad.
El sistema es modular y permite ampliaciones futuras (pantalla, WiFi, registro de datos, control remoto).

---

# Fundamentación pedagógica
El proyecto está pensado para nivel secundario técnico, terciario y espacios maker.
Integra contenidos de electrónica, programación, robótica, sustentabilidad y fabricación digital.
Promueve el aprendizaje basado en proyectos (ABP) y el trabajo interdisciplinario.
